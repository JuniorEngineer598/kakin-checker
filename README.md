# 課金チェッカー

ソシャゲやサブスクなどのアプリ内課金を記録・管理するためのWebアプリです。

月パス・ガチャ石・お得パックなど、複数のアプリに分かれた課金履歴を記録し、月ごとの支出やアプリ別の課金状況を確認できます。

## デモ

[課金チェッカーを開く](https://kakin-checker-raxi.vercel.app)

## 作成した理由

複数のソシャゲを遊んでいると、月パス・ガチャ石・お得パックなどの課金がアプリごとに分散し、今月いくら使ったのか分かりにくいと感じました。

そこで、課金内容を簡単に記録し、月別・アプリ別に支出を可視化できるWebアプリを作成しました。

## 主な機能

- ユーザー認証
- Googleログイン
- アプリ情報の追加・管理
- アプリアイコンのアップロード・変更
- 課金記録の追加・管理
- テンプレートからの課金記録追加
- 課金履歴の一覧表示
- アプリごとの課金額管理
- カレンダーでの課金日の確認
- 月別課金額の分析
- アプリ別課金割合の表示

## データ保存について

- このアプリでは、Supabase AuthとSupabase Databaseを利用して、ログインユーザーごとにデータを保存しています。

- データは主に以下の3つに分けて管理しています。

- `apps`：ユーザーが登録したアプリ情報
- `charges`：課金金額・日付・種類・メモなどの課金記録
- `charge_templates`：よく使う課金内容を保存するテンプレート

- 課金記録はアプリ情報と紐づけることで、アプリごとの合計金額や月ごとの課金額を集計しやすい構成にしています。

- また、各データにはログインユーザーのIDを紐づけ、ユーザーごとにデータを分けて扱えるようにしています。

- セキュリティ面では、SupabaseのRLS（Row Level Security）を設定し、認証済みユーザーが自分に紐づくデータだけを操作できるようにしています。
