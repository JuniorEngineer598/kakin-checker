# 課金チェッカー

ソシャゲやアプリ内課金を記録・管理するためのWebアプリです。

## デモ

[課金チェッカーを開く](https://kakin-checker-raxi.vercel.app)

## 主な機能

- 課金記録の追加・管理
- テンプレートからの課金記録追加
- 課金履歴の一覧表示
- アプリごとの課金額管理
- カレンダーでの課金日の確認
- 月別課金額の分析
- localStorageによるデータ保存

## 工夫した点

- よく使う課金内容をテンプレート化し、ワンクリックで課金記録を追加できるようにしました。
- ホーム画面では、今月の課金額や課金件数、アプリ別の課金状況を一目で確認できるようにしました。
- 課金額・件数・履歴・カレンダー・分析を画面ごとに整理し、必要な情報をすぐ確認できる構成にしました。
- シンプルなデザインを意識し、スマホでも見やすく操作しやすいUIにしました。

## 技術スタック

- Next.js
- React
- TypeScript
- Tailwind CSS
- localStorage

## データ保存について

このアプリは現在、ブラウザのlocalStorageにデータを保存します。
ログインやクラウド同期はありません。
