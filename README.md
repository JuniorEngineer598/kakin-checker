# 課金チェッカー

ソシャゲやアプリ内課金を記録・管理するためのWebアプリです。

月パス・ガチャ石・お得パックなど、複数のアプリに分かれた課金履歴を記録し、月ごとの支出やアプリ別の課金状況を確認できます。

## デモ

[課金チェッカーを開く](https://kakin-checker-raxi.vercel.app)

## 作成した理由

複数のソシャゲを遊んでいると、月パス・ガチャ石・お得パックなどの課金がアプリごとに分散し、今月いくら使ったのか分かりにくいと感じました。

そこで、課金内容を簡単に記録し、月別・アプリ別に支出を可視化できるWebアプリを作成しました。

## 主な機能

- ユーザー認証
- Googleログイン
- アプリ情報の追加・管理
- 課金記録の追加・管理
- テンプレートからの課金記録追加
- 課金履歴の一覧表示
- アプリごとの課金額管理
- カレンダーでの課金日の確認
- 月別課金額の分析
- アプリ別課金割合の表示
- Supabaseによるユーザーごとのデータ保存

## 工夫した点

- よく使う課金内容をテンプレート化し、ワンクリックで課金記録を追加できるようにしました。
- ホーム画面では、今月の課金額や課金件数、アプリ別の課金状況を一目で確認できるようにしました。
- 課金額・件数・履歴・カレンダー・分析を画面ごとに整理し、必要な情報をすぐ確認できる構成にしました。
- PCだけでなく、スマホでも見やすく操作しやすいUIを意識しました。

## データ保存について

現在は、ブラウザのlocalStorageにデータを保存しています。

そのため、ログイン機能やクラウド同期はまだありません。  
今後は認証機能とデータベース保存を追加し、ユーザーごとに課金データを管理できるようにする予定です。
